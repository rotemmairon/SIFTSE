# SIFTSE

This code is a Matlab implementation for the model suggested in "Semantically-based human scanpath estimation with hmms" (Liu, Huiying, Dong Xu, Qingming Huang, Wen Li, Min Xu, and Stephen Lin, In ICCV 2013).
